# Linux Networking Practice

This repository contains solutions and practical exercises for Linux Networking. The project covers fundamental networking concepts, Linux network configuration, connectivity testing, packet analysis, firewall management, DNS configuration, routing analysis, protocol comparison, and network performance evaluation.

Each exercise is organized in a separate directory and includes the required scripts, configuration files, reports, and documentation.

## Repository Structure

- Exercise 01 – Network Information Collection
- Exercise 02 – Static IP Configuration using Netplan
- Exercise 03 – Connectivity Testing
- Exercise 04 – Port Investigation
- Exercise 05 – Packet Capture with tcpdump
- Exercise 06 – PCAP Analysis using tshark
- Exercise 07 – Port Accessibility Testing
- Exercise 08 – OSI Model and Troubleshooting
- Exercise 09 – Firewall Configuration
- Exercise 10 – DNS Configuration
- Exercise 11 – Routing Analysis
- Exercise 12 – TCP vs UDP Analysis
- Exercise 13 – Network Performance Testing
